Array is not growble (means array have fixed size) in nature.
We can store only homogenous data type in array
Array don't have inbuilt method. For that we have to return programming logic

to overcome these problems we have collections in java.

Collection :
- growble in nature
- homogenous + heterogenous data
-underlaying data structure (redymade methods are available)

Array : 
-not growble (Fixed size)
- homogenous data
-no underlaying Data Structure (redimade method are not available)



Interfaces(I) : 

Collection(I) : Root interface 
*******************************

Methods : 
- add(Object o);
- addAll(Collection c) //you can add multiple object at time
- remove(Object o)
- removeAll(Collection c)
- retailAll(Collection c) //except this group of object other object will be deleted
- clear();
- isEmpty();
- size();
- contains(Object o); //Particular object is present or not
- containsAll(Collection c); //check group of object is present or not 
- toArray(Collection c);	//convert array into array format. It will return object

1) List(I) :
============= 
- Child of Collection(I)
- follow insertion order
- duplicates allowed

Methods : 
- All methods of Collection(I)
- add(index, Object o);	//add object in specific position
- addAll(index, Collection c);
- remove(index);
- get(index);	//retrive object
- set(index, Object o); //update object given index

Implemented classes : 
---------------------
I) ArrayList(C) :
- present in java.util package
- Default memory size is 10  
	
II) Linked List
III) Vector => Stack (legacy class)


2) Set(I) : 
- child interface of collection(I)
- insertion order not preserved
- duplicates not allowed
	I) HashSet
	II) LinkedHashSet 


3) Queue(I) : 
- child interface of Collection(I)
- prior to processing (FIFO - First in First Out)
- ex., If u want to send multiple email to customer then here we can prefer Queue
	I) PriorityQueue


Map(I) : Indidual or seprate interface. It is under the Collection Framerwork but ont in Collection(I)
- if u want to store data in the form of key and value pair
- keys duplicate are not allowed but value duplicate are allowed
	I) HashMap
	II) LinkedHashMap
	III) HashTable
